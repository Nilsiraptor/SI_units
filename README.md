# SI Units
A Python Module for physics calculations with the international standard of units.

It includes the following mathematical and physical constants:
- Speed of Light
- Planck Constant
- Elementary Charge
- Boltzmann Constant
- Avogadro Constant
- Luminous Efficancy
- Hyperfine Transition Frequency of Cs-133
- Pi
- Euler's Number
- Tau
- Golden Ratio
